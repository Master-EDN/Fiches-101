Exporter vers Anki

Le fichier exporté est un .txt séparé par tabulations (Front / Back / Tags), au format attendu par Anki. Les images ne sont jamais incluses — seul le texte (question et correction) est exporté.

Dans Anki : Fichier → Importer, sélectionne le fichier, et vérifie que le séparateur de champs est bien "Tabulation" et que la case "Le premier champ contient les tags" n'est pas cochée (les tags sont dans une colonne dédiée).
